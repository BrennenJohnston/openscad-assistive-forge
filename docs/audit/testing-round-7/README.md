## Re-Running the CLI Harness

### Prerequisites

1. **OpenSCAD 2021.01** installed at `C:\Program Files\OpenSCAD\`
2. **OpenSCAD Nightly** installed at `C:\Program Files\OpenSCAD (Nightly)\`
3. **Node.js** (>= 20) available in PATH
4. Test fixture at `tests/fixtures/keyguard-v75/` (committed to repo)

### Commands

```powershell
# Dry run (prints commands without executing OpenSCAD)
powershell -ExecutionPolicy Bypass -File scripts/desktop-audit.ps1 -DryRun

# Full run (generates reference data)
powershell -ExecutionPolicy Bypass -File scripts/desktop-audit.ps1

# Custom output directory
powershell -ExecutionPolicy Bypass -File scripts/desktop-audit.ps1 -OutputDir "path/to/output"
```

The script:
1. Validates both OpenSCAD installations are reachable
2. Copies the test fixture to a temp working directory
3. For each (version x scenario): runs CLI exports, captures console, parses geometry stats, extracts COFF face colors (Nightly only)
4. Writes structured JSON to the output directory
5. Prints a summary comparison table to stdout
6. Preserves the temp directory for inspection (prints cleanup command)

Expected runtime: 2-3 minutes (dominated by CGAL rendering).

### OFF Color Extraction (standalone)

```powershell
node scripts/parse-off-colors.js path/to/file.off
```

Reads an OFF/COFF file and outputs a JSON object with `is_coff`, `total_faces`,
and `unique_colors` (each with `rgb`, `hex`, `css_name`, `face_count`).

## Extending With Additional Scenarios

To add new test scenarios:

1. **Edit `scripts/desktop-audit.ps1`** -- Add entries to the `$Scenarios` array:

```powershell
@{
    id         = "new-scenario-id"
    params     = @{ type_of_keyguard = "..."; generate = "..." }
    geom_type  = "3D"       # or "2D"
    svg_export = $false     # $true for 2D scenarios
}
```

2. **Re-run the harness** -- New JSON files will appear in the output directory.

3. **Update `desktop-baseline-v75.md`** -- Add a new scenario section following
   the existing structure.

To add a new OpenSCAD version, add an entry to the `$Versions` array in the
script and re-run.
